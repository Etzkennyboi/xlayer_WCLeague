# X Cup Manager — Web3 Football

![X Cup Manager](https://img.shields.io/badge/Network-X_Layer_Testnet-cyan.svg) ![Framework](https://img.shields.io/badge/Framework-Next.js_16-black.svg) ![Web3](https://img.shields.io/badge/Web3-Ethers.js_v5-blue.svg)

**X Cup Manager** is the world's first AAA Web3 Football Manager, Fantasy League, and Prediction Market built natively on **X Layer Testnet**. 

This application seamlessly bridges immersive, premium football gameplay (like interactive Penalty Shootouts and live match simulations) with high-stakes, decentralized prediction markets where users can stake OKB directly from their MetaMask wallets.

---

## 🌟 Core Features

### 1. Decentralized Prediction Market (Matchday)
- **On-Chain Betting:** Users can connect their MetaMask wallets and stake real OKB on upcoming 2026 World Cup fixtures.
- **Dynamic Odds & Pools:** Staked OKB goes into a decentralized liquidity pool (Home/Draw/Away). Winning bets take a proportional split of the total pool once the market is resolved.
- **Oracle Resolution:** The app features a robust Node.js backend acting as an Oracle. Admins can securely resolve match outcomes via the Oracle Dashboard, which broadcasts the results directly to the smart contract on X Layer.

### 2. High-Stakes PK Shootout Mini-Game
- **Real Stakes:** Users must stake a minimum OKB equivalent (e.g., $1) to play the Penalty Shootout mini-game.
- **AI-Powered Goalkeeper:** The game integrates with DeepSeek AI to track the user's shooting history and adapt the goalkeeper's dive patterns, creating a genuinely challenging Web3 gaming experience.
- **Automated Payouts:** Winning the shootout triggers an automatic, gasless backend payout of the prize money directly back to the user's wallet via the Agentic Treasury.
- **AI Commentary:** Receives real-time, dynamic commentary generated by AI based on the user's performance.

### 3. NFT Achievement Collection
- **On-Chain Badges:** Completing milestones (like winning matches or specific tournaments) allows the user to mint exclusive Achievement NFT Badges directly to their wallet.
- **Gas-Optimized:** Backend minting logic is prepared to handle gasless NFT distribution to reduce friction for new Web3 users.

### 4. Fantasy League & Live Simulations
- **Team Management:** Select your favorite World Cup team and manage them through the tournament bracket.
- **Live Match Engine:** A butter-smooth, lag-free match simulation engine built with heavily memoized React components that renders live match events, goals, and commentary.
- **Global Leaderboard:** Ranks players across the platform based on their Fantasy points.

## 🔗 Web3 Integration Details

X Cup Manager makes heavy use of advanced Web3 patterns to provide a seamless Web2-like experience:

- **Network:** OKX X Layer Testnet (MetaMask Chain ID: `195`)
- **Native Token:** OKB is used for all transactions, bets, and payouts.
- **Smart Contracts:**
  - `PredictionMarket.sol`: Handles market creation, user bets, pool calculations, and secure claims.
  - `WorldCupNFT.sol`: Standard ERC721 for achievement badges.
- **Hybrid Oracle Architecture:** 
  - To bypass Account Abstraction (AA) wallet rate-limits and nonce collisions, the backend uses a mixture of pure Ethers.js wallet signing (`scripts/init_markets.cjs`) for bulk operations, and `onchainos` CLI integrations for targeted single-resolution Oracle updates.
  - The API endpoints (e.g., `/api/fixtures`) dynamically poll the X Layer RPC directly via `eth_call` to prevent Next.js server-side `fetch` caching bugs, ensuring real-time synchronized odds on the frontend.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- MetaMask extension installed in your browser with OKB Testnet tokens.

### Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Environment Variables:**
   Create a `.env.local` or `.env` file in the root directory and ensure you have the required keys:
   ```env
   NEXT_PUBLIC_PREDICTION_MARKET_ADDRESS=0x25b82B010f5FC38417d1d70f62B0f7b37cB3eCc5
   NEXT_PUBLIC_TREASURY_ADDRESS=0x491d07ede06eebf25a3a0ff3ce3b78ca78af4aac
   # Add your deployer private key if you intend to run the oracle scripts locally
   NEXT_PUBLIC_MINTER_PK=your_private_key_here
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the application.

4. **Production Build:**
   The application is optimized for Vercel. To test the production build locally:
   ```bash
   npm run build
   npm run start
   ```

## 📜 Smart Contract Deployment

If you need to deploy fresh contracts to the X Layer Testnet:

1. **Compile Contracts:**
   ```bash
   npx hardhat compile
   ```

2. **Deploy directly via script:**
   ```bash
   node scripts/deploy_direct.cjs
   ```
   *(Ensure your `deploy_direct.cjs` is configured with a funded private key).*

3. **Initialize Prediction Markets:**
   Once deployed, seed the initial World Cup fixtures onto the blockchain:
   ```bash
   node scripts/init_markets.cjs
   ```
